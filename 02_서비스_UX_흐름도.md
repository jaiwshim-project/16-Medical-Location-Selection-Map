# 02. 서비스 전체 UX 흐름도

## A️⃣ 사용자 여정 전체 맵

```
┌─────────────────────────────────────────────────────────────────────┐
│                    치과 개업 의사결정 맵 — 사용자 여정               │
└─────────────────────────────────────────────────────────────────────┘

[1단계: 랜딩]          [2단계: 프로필]        [3단계: 탐색]
 홈페이지 방문    →     회원가입/로그인   →    지도 탐색
 서비스 소개             원장 유형 설문          지역 검색
 무료 체험 시작          예산·선호 입력          히트맵 보기

        ↓                      ↓                     ↓

[4단계: 분석]          [5단계: 비교]         [6단계: 결정]
 입지 상세 분석   →     개업 vs 인수 비교  →   AI 결정 리포트
 투자금 시뮬레이션       후보지 비교              추천·리스크
 경쟁 분석               점수 대시보드            액션 플랜

        ↓                      ↓                     ↓

[7단계: 실행]          [8단계: 연결]         [9단계: 사후]
 리포트 PDF 다운  →     전문가 매칭      →    개원 후 모니터링
 체크리스트              대출·금융 연계          상권 변화 알림
 타임라인                인테리어·장비           경쟁 현황 추적
```

## 상세 화면 흐름 (Screen Flow)

### Phase 1: 온보딩

```
[S1. 랜딩 페이지]
  │
  ├─→ "무료로 시작하기" 버튼
  │     │
  │     ▼
  │   [S2. 회원가입]
  │     - 이메일/비밀번호
  │     - 또는 카카오/네이버 소셜 로그인
  │     │
  │     ▼
  │   [S3. 원장 프로필 설문 (5단계)]
  │     Step 1: "현재 상태가 어떻게 되시나요?"
  │             ○ 수련 중 (인턴/레지던트)
  │             ○ 봉직의 (페이닥터)
  │             ○ 경력 치과의사 (5년+)
  │             ○ 전문의 (교정/구강외과 등)
  │
  │     Step 2: "어떤 개원을 생각하고 계신가요?"
  │             ○ 신규 개원
  │             ○ 기존 치과 인수
  │             ○ 아직 결정 못 함 → 둘 다 분석
  │
  │     Step 3: "예산 범위를 알려주세요"
  │             슬라이더: 1억 ───●─── 10억+
  │             □ 대출 포함    □ 자기자본만
  │
  │     Step 4: "관심 지역을 선택해 주세요"
  │             지도에서 시/구 단위 다중 선택
  │             또는 "전국 어디든" 체크
  │
  │     Step 5: "전문 분야를 선택해 주세요"
  │             □ 일반진료  □ 임플란트  □ 교정
  │             □ 심미보철  □ 소아치과  □ 기타
  │
  ├─→ "로그인 없이 둘러보기"
        │
        ▼
      [S4. 제한된 지도 뷰] (블러 처리된 상세 데이터)
```

## 네비게이션 구조

```
홈 (/)
├── 서비스 소개 (/about)
├── 요금제 (/pricing)
├── 로그인 (/login)
│
├── [로그인 후]
│   ├── 대시보드 (/dashboard)
│   │   ├── 내 프로필 (/profile)
│   │   ├── 관심 목록 (/favorites)
│   │   └── 내 리포트 (/reports)
│   │
│   ├── 지도 탐색 (/map)
│   │   ├── 히트맵 뷰 (/map?view=heatmap)
│   │   ├── 클러스터 뷰 (/map?view=cluster)
│   │   └── 위성 뷰 (/map?view=satellite)
│   │
│   ├── 입지 분석 (/analysis/:dong_code)
│   │   ├── 인구 탭 (/analysis/:id/population)
│   │   ├── 경쟁 탭 (/analysis/:id/competition)
│   │   ├── 비용 탭 (/analysis/:id/cost)
│   │   ├── 접근성 탭 (/analysis/:id/access)
│   │   └── 종합 탭 (/analysis/:id/summary)
│   │
│   ├── 투자 시뮬레이터 (/simulator)
│   ├── 후보지 비교 (/compare)
│   ├── AI 결정 리포트 (/report/:id)
│   │
│   ├── 매물 탐색 (/listings)
│   │   ├── 인수 매물 (/listings/acquisition)
│   │   └── 임대 매물 (/listings/lease)
│   │
│   └── 전문가 연결 (/experts)
│       ├── 금융 상담 (/experts/finance)
│       ├── 인테리어 (/experts/interior)
│       └── 세무/법률 (/experts/legal)
│
├── 블로그/인사이트 (/blog)
├── FAQ (/faq)
└── 문의하기 (/contact)
```
