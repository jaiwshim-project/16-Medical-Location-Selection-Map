### 무료→유료 전환 트리거

```
무료 사용자가 유료로 전환되는 핵심 순간:

1. "이 지역 상세 점수가 궁금한데…"
   → 무료는 종합 점수만, 5축 분석은 베이직부터
   → 블러 처리된 상세 점수 + "잠금 해제" 버튼

2. "이 자리에 얼마가 필요한지 알고 싶다"
   → 투자 시뮬레이션은 베이직부터
   → "투자금 계산하기" 클릭 → 요금제 안내

3. "PDF로 뽑아서 은행에 보여줘야 하는데"
   → AI 리포트 PDF는 베이직부터
   → "리포트 생성" → 무료 한도 초과 시 업그레이드 유도

4. "여기랑 저기랑 비교해 보고 싶다"
   → 후보지 비교는 베이직부터
```

```

✅ 랜딩 페이지 전체

✅ 블로그/인사이트 콘텐츠
✅ FAQ
✅ 전국 지도 탐색 (시도 단위 히트맵)

볼 수 없는 것:
🔒 행정동 단위 상세 데이터
🔒 입지 분석 점수
🔒 투자 시뮬레이터
```

## 4️⃣ MVP 기준 최소 홈페이지 구성안

### MVP 포함 페이지 (5개)

```
1. 랜딩 페이지 (/)
   - Hero + 문제제기 + 기능 소개 + CTA
   - 단일 페이지로 구성 (원페이지)

2. 회원가입/로그인 (/auth)
   - 이메일 + 카카오 소셜 로그인
   - 5단계 프로필 설문

3. 메인 대시보드 — 지도 뷰 (/map)
   - 히트맵 (인구밀도, 경쟁강도)
   - 치과 위치 마커
   - 클릭 → 입지 요약 사이드패널

4. 입지 분석 페이지 (/analysis/:id)
   - 5축 점수 + 상세 문장
   - 투자 시뮬레이션 (기본)
   - AI 리포트 생성

5. 요금제 & 결제 (/pricing)
   - 3-Tier 요금제
   - Stripe/토스페이먼츠 결제

MVP 제외 (Phase 2):
- 후보지 비교 대시보드
- 인수 매물 탐색
- 전문가 매칭
- 블로그/인사이트
- 개원 후 모니터링
```
