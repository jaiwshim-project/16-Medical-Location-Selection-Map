## 2️⃣ 특허 출원용 서비스 전체 구조 요약본

### [발명의 명칭]
**"복합 데이터 기반 의료기관 개원 입지 분석 및 의사결정 지원 시스템"**

### [기술 분야]
본 발명은 공공데이터 및 상업데이터를 융합하여 의료기관(특히 치과) 개원 입지의 적합도를 다축 점수로 산출하고, 사용자 프로필에 기반한 맞춤형 추천과 AI 기반 의사결정 리포트를 자동 생성하는 시스템 및 방법에 관한 것이다.

### [발명이 해결하고자 하는 과제]
1. 의료기관 개원 시 입지 선정에 객관적 데이터 분석 도구의 부재
2. 신규 개원과 기존 기관 인수 간 경제성 비교의 어려움
3. 개원 의사의 개인 조건(경력, 예산, 전문 분야)에 맞는 맞춤형 추천 부재
4. 투자금 산출과 수익 전망에 대한 체계적 시뮬레이션 도구의 부재

### [청구항 (요약)]

```
[청구항 1]
복수의 공공 데이터 소스로부터 인구통계, 의료기관 현황, 상권 정보,
부동산 정보, 교통 정보를 수집하는 데이터 수집 단계;
수집된 데이터를 지리 좌표 기준으로 정규화하고
공간 분석을 수행하는 데이터 처리 단계;
사용자로부터 경력, 예산, 전문 분야, 선호 지역을 포함하는
프로필 정보를 입력받는 프로필 수집 단계;
상기 프로필 정보에 기반하여 사용자 유형을 분류하고,
유형별 가중치를 설정하는 사용자 분류 단계;
인구/수요, 경쟁 환경, 비용 효율, 접근성, 성장성의 5개 축에 대해
각각 점수를 산출하고, 상기 유형별 가중치를 적용하여
종합 점수를 산출하는 다축 점수 산출 단계;
상기 종합 점수와 개별 축 점수에 기반하여,
점수 구간별 문장 템플릿을 선택하고 사용자 특성에 맞게
어조를 조정하여 자연어 리포트를 생성하는 리포트 생성 단계;
를 포함하는 의료기관 개원 입지 분석 방법.

[청구항 2]
청구항 1에 있어서,
동일 지역에 대해 신규 개원 시나리오의 점수와
기존 기관 인수 시나리오의 점수를 각각 산출하고,
두 점수의 차이와 사용자 유형별 편향치를 적용하여
개원 또는 인수 중 하나를 추천하는 비교 판정 단계를
추가로 포함하는 의료기관 개원 입지 분석 방법.

[청구항 3]
청구항 1에 있어서,
상기 리포트 생성 단계는,
축별 점수의 표준편차가 기준값 이상일 때 문장 비중을 높이고,
총점이 기준 구간 밖일 때 점수 비중을 높이는
하이브리드 조합 규칙을 적용하는 것을 특징으로 하는
의료기관 개원 입지 분석 방법.
```

### [발명의 효과]
1. 주관적 판단에 의존하던 개원 입지 선정을 객관적 데이터 기반으로 전환
2. 사용자 유형별 맞춤 가중치 적용으로 개인화된 추천 제공
3. 신규 개원과 인수의 동시 비교를 통한 합리적 의사결정 지원
4. 점수와 자연어의 하이브리드 결합으로 직관성과 맥락성 동시 확보
