# 10. 추가 고려사항 및 확장 전략

---

## 기획 시 추가로 고려해야 할 항목들

### A. 법률·규제 리스크

| 항목 | 리스크 | 대응 |
|------|--------|------|
| 의료법 | 의료기관 개설 관련 정보 제공이 "의료 광고"로 해석될 가능성 | 면책 조항 필수, 법률 검토 |
| 공공데이터 상업 이용 | 일부 공공 API의 상업적 이용 제한 조건 | 이용약관 사전 확인 |
| 개인정보보호법 | 원장 프로필, 예산 정보 = 민감 개인정보 | 개인정보처리방침, 동의 절차 |
| 부동산 중개법 | 매물 정보 제공이 "중개"로 해석될 가능성 | 정보 제공만 (중개 행위 금지) |
| 크롤링 법적 리스크 | 네이버 플레이스 등 크롤링 | robots.txt 준수, API 우선 |
| 금융소비자보호법 | 대출 비교/추천이 "대출 중개"로 해석될 수 있음 | 대출모집인 등록 또는 정보 제공만 |

### B. 데이터 정확도 & 신뢰도

```
문제: 공공 데이터는 6개월~1년 지연
대응:
  1. 데이터 기준일 항상 명시
  2. "실시간"이라 표현하지 않음
  3. 여러 소스 교차 검증 (삼각검증)
  4. 정확도 한계 사전 고지

문제: 점수 모델의 신뢰성 검증
대응:
  1. 실제 개원 사례와 점수의 상관관계 분석 (β 테스트)
  2. 전문가 패널 리뷰 (치과 컨설턴트 3~5명)
  3. 모델 버전 관리 및 정확도 추적
  4. 사용자 피드백 루프 (개원 결과 추적)
```

### D. 마케팅·고객 획득 전략

```
Phase 1 (베타): 초기 사용자 100명 확보
  - 치과의사 온라인 커뮤니티 (메디게이트, 치과의사신문)
  - 치과대학 동문회 타겟 홍보
  - 베타 테스터 무료 + 피드백 리워드
  - 치과 수련병원 세미나 참가

Phase 2 (런칭): 월 100명 가입
  - 네이버/구글 키워드 광고 ("치과 개원", "치과 입지 분석")
  - 블로그 콘텐츠 마케팅 (SEO)
  - 치과 장비 전시회 부스
  - 치과 컨설턴트와 제휴

Phase 3 (성장): 월 300명 가입
  - 성공 사례 레퍼런스 마케팅
  - 추천인 프로그램 (1개월 무료)
  - 치과의사협회 파트너십
  - 금융사 공동 마케팅
```

### E. 확장 전략

```
수직 확장 (같은 타겟, 더 깊게):
  - 개원 후 경영 분석 대시보드
  - 환자 마케팅 도구 (지역 타겟 광고)
  - 직원 채용 플랫폼 연계
  - 의료 기기 마켓플레이스

수평 확장 (같은 모델, 다른 업종):
  - 의과 (내과, 피부과, 정형외과 등)
  - 한의과, 약국
  - 동물병원
  - 안경점, 보청기 전문점
  → "메디맵(MediMap)" 통합 플랫폼

지역 확장:
  - 일본 (치과 시장 규모 3배)
  - 동남아 (베트남, 태국 — 급성장)
```

### F. 팀 구성 (MVP 최소)

```
필수 (3명):
  - PM/기획자 1명: 기획, 데이터 소싱, 사업
  - 풀스택 개발자 1명: Next.js + PostgreSQL + AI 연동
  - 디자이너 1명: UI/UX, 와이어프레임, 디자인 시스템

권장 추가 (2명):
  - 데이터 엔지니어 1명: ETL, 공공 API, 데이터 정합성
  - 마케터 1명: 콘텐츠, SEO, 커뮤니티

도메인 어드바이저 (파트타임):
  - 치과 원장 1~2명 (실무 피드백)
  - 부동산/상권 분석 전문가 1명
```

### G. 리스크 & 마일스톤

```
핵심 리스크:
  1. 데이터 정확도 → 전문가 검증 + 사용자 피드백
  2. 유료 전환율 저조 → 무료→유료 트리거 최적화
  3. 경쟁 서비스 출현 → 특허 + 데이터 해자 + 빠른 실행
  4. 법적 리스크 → 사전 법률 검토 (변호사)
  5. 개발 지연 → MVP 범위 축소 (핵심만)

마일스톤:
  M1: 데이터 파이프라인 완성 (4주)
  M2: 지도 + 히트맵 동작 (6주)
  M3: 점수 엔진 + AI 리포트 (8주)
  M4: 결제 + 유료 기능 (10주)
  M5: 베타 출시 (10주)
  M6: 유료 고객 50명 (출시 후 2개월)
  M7: BEP 달성 (출시 후 6~12개월)
```
