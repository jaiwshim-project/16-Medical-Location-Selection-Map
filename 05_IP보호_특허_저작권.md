# 05. 지식재산(IP) 보호 전략 — 특허·저작권·영업비밀

---

## 복제 위험 분석

### 공개 시 복제 가능한 부분

| 구분 | 복제 난이도 | 보호 가능 여부 |
|------|-----------|--------------|
| UI/UX 디자인 | 높음 (쉽게 모방) | △ 디자인 등록 |
| 기능 아이디어 (상권분석 맵) | 높음 | × 아이디어 자체는 보호 불가 |
| 데이터 수집·가공 파이프라인 | 중간 | ○ 영업비밀 |
| 스코어링 알고리즘 | 중간 | ○ 특허 + 영업비밀 |
| AI 리포트 생성 로직 | 중간 | ○ 특허 + 저작권 |
| 누적 데이터·학습 모델 | 낮음 (시간 필요) | ○ 영업비밀 + DB권 |
| 카피라이팅·문장 템플릿 | 높음 | ○ 저작권 |
| 브랜드·서비스명 | — | ○ 상표권 |

### 핵심 방어 전략: **데이터 + 알고리즘 + 네트워크 효과**

```
경쟁사가 복제하기 어려운 3대 해자(Moat):

1. 누적 데이터 (시간이 만드는 장벽)
   - 실제 개원 결과 피드백 데이터
   - 사용자 행동 데이터 (어떤 입지를 선호/기피하는지)
   - 매물 히스토리 (가격 변동, 거래 이력)

2. 알고리즘 정교도 (경험이 만드는 장벽)
   - 점수→실제 결과 피드백 루프로 모델 개선
   - 원장 유형별 추천 정확도 향상
   - 지역별 미세 가중치 튜닝

3. 네트워크 효과 (규모가 만드는 장벽)
   - 더 많은 원장 → 더 많은 데이터 → 더 정확한 분석
   - 금융사·장비업체·인테리어 제휴 네트워크
   - 커뮤니티·후기·사례 축적
```

## 특허 출원 전략

### 출원 가능한 발명 (BM 특허)

#### 특허 1: 치과 개원 입지 점수 산출 시스템

```
[명칭]
"복수의 공공 데이터를 융합하여 치과 의원 개원 적합도를
점수화하는 방법 및 시스템"

[청구항 핵심]
- 인구통계, 경쟁현황, 부동산, 교통, 상권 데이터를 수집하는 단계
- 사용자 프로필에 따라 가중치를 동적으로 조정하는 단계
- 5대 축 기반 종합 점수를 산출하는 단계
- 점수에 기반하여 추천 입지를 랭킹하는 단계

[분류] G06Q 50/22 (의료 서비스) + G06Q 30/02 (상거래 분석)
```

#### 특허 2: 개업 vs 인수 비교 판정 시스템

```
[명칭]
"의료기관 신규 개원과 기존 기관 인수의 경제성을
비교 분석하여 의사결정을 지원하는 시스템 및 방법"

[청구항 핵심]
- 동일 지역의 신규 개원 시뮬레이션 데이터와
  인수 매물 데이터를 동시에 수집하는 단계
- 투자 회수 기간, 리스크, 성장성을 비교 점수화하는 단계
- 사용자 유형별 가중치를 적용하여 판정하는 단계
- AI 기반 자연어 판정 리포트를 생성하는 단계
```

#### 특허 3: AI 기반 개원 결정 리포트 생성 시스템

```
[명칭]
"점수 데이터와 조건 분기를 조합하여 맞춤형 개원 의사결정
리포트를 자동 생성하는 인공지능 시스템 및 방법"

[청구항 핵심]
- 복수의 분석 점수를 입력받는 단계
- 점수 조합 규칙에 따라 문장 템플릿을 선택하는 단계
- 사용자 유형·리스크 허용도에 따라 어조를 조정하는 단계
- 점수와 자연어를 하이브리드로 조합하여 리포트를 출력하는 단계
```

### 특허 출원 비용 및 일정

```
국내 출원 (건당):
- 변리사 수임료: 200~400만원
- 출원료: 약 5만원
- 심사청구료: 약 15만원
- 등록료: 연차별 (1~3년 약 5만원/년)
- 소요기간: 12~18개월

3건 출원 예상 비용: 700~1,300만원

PCT 국제출원 (선택):
- 추가 300~500만원/건
- 미국/일본 진입 시 각 500~800만원 추가
```

### 출원 시기 전략

```
Phase 1 (즉시): 국내 우선출원 3건
  → 아이디어 공개 전에 출원일 확보 (출원일이 핵심!)

Phase 2 (12개월 내): PCT 국제출원 여부 결정
  → 해외 시장 진출 계획에 따라

Phase 3 (30개월 내): 주요국 진입
  → 미국, 일본 등 목표 시장
```

|---|--------|------|----------|

```
대리 등록 (변리사): 건당 30~50만원

7건 패키지 예상: 250~400만원 (대리 포함)
```

## 상표권 등록

```
서비스명 상표 등록:
- 출원료: 약 6만원 (전자출원)
- 등록료: 약 21만원 (10년)
- 변리사 수임: 30~50만원

상표 분류:
- 제9류: 소프트웨어, 앱
- 제35류: 데이터 분석 서비스, 사업 컨설팅
- 제36류: 부동산 정보, 금융 서비스
- 제42류: SaaS, 클라우드 서비스
- 제44류: 의료 정보 제공

4개 류 출원 예상: 150~250만원
```

## 영업비밀 보호

### 영업비밀로 관리할 항목

```
1. 스코어링 가중치의 구체적 수치
2. 데이터 전처리·정규화 로직
3. AI 모델 파인튜닝 파라미터
4. 사용자 행동 데이터 분석 결과
5. 제휴사별 수수료율·계약 조건
6. 매물 검증 프로세스 상세
7. 고객 세그먼트별 전환율 데이터
```

### 보호 조치 (법적 요건)

```
□ 비밀 표시 ("대외비", "Confidential" 마킹)
□ 접근 권한 제한 (Need-to-know 기반)
□ NDA (비밀유지계약) — 직원, 외주, 제휴사
□ 기술적 보호 (암호화, 접근 로그)
□ 비밀관리 규정 사내 문서화
□ 퇴직자 비밀유지 의무 서약
```
