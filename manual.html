서비스 개요

덴탈맵AI는 치과 개원을 준비하는 치과의사를 위한 데이터 기반 입지 분석 플랫폼입니다. 건강보험심사평가원(HIRA)의 전국 치과 실제 데이터(약 19,500개)와 카카오 지오코딩을 활용하여 인구/수요, 경쟁 환경, 비용 효율, 접근성, 성장성을 5축으로 종합 분석하고, AI가 최적의 개원 위치 3곳을 추천합니다.

핵심 기능

  • 01카카오 지오코딩 검색 — '강남역', '인천경찰청' 등 구체적 장소명을 입력하면 정확한 좌표를 자동 변환
  • 02HIRA 실제 치과 데이터 — 전국 19,500+ 치과의원·치과병원의 위치, 의사 수, 전문의 수, 개설일 등 실제 데이터 활용
  • 035축 입지분석 — 인구/수요(25%), 경쟁(25%), 비용(20%), 접근성(15%), 성장성(15%)의 가중 분석
  • 04AI 추천 개원 위치 3곳 — 밀집도×틈새(density×gap) 알고리즘으로 검증된 상권 내 최적 위치 추천
  • 05시작점 이동 & 재분석 — 지도 마커를 드래그하여 원하는 위치에서 즉시 재분석
  • 06AI 입지분석 리포트 — 9개 섹션 종합 보고서, 새 창에서 확인 및 PDF 출력 (페이지 번호 포함)
  • 07리포트 목록 관리 — 생성한 리포트를 자동 저장하고 목록에서 다시 열람
  • 08투자 시뮬레이션 — 초기 투자금, 월 비용, 예상 손익 시뮬레이션

데이터 출처

건강보험심사평가원(HIRA) — 병원정보서비스 API를 통해 전국 치과의원·치과병원 데이터(GPS 좌표, 의사 수, 전문의 수, 개설일 등)를 매주 자동 갱신합니다.

카카오 Local API — 키워드 검색·지오코딩을 통해 사용자가 입력한 장소명을 정확한 GPS 좌표로 변환합니다.

기타 공공 데이터 — 통계청(인구), 국토교통부(부동산), 소상공인진흥공단(상권) 등 공공기관 공식 데이터를 참고합니다.

시작하기

덴탈맵AI는 별도의 설치 없이 웹 브라우저에서 바로 사용할 수 있습니다.

빠른 시작 가이드

덴탈맵AI 홈페이지에 접속합니다. 테스트 모드에서는 로그인 없이 모든 기능을 체험할 수 있습니다.

2

관심 지역 검색

상단 검색창 또는 입지분석 메뉴에서 분석하고 싶은 지역이나 장소를 검색합니다. 예: 「강남역」, 「인천경찰청」, 「수원 영통구」, 「판교 현대백화점」 등. 카카오 지오코딩으로 정확한 위치를 자동 변환합니다.

종합 점수, 5축 분석, HIRA 실제 치과 데이터(반경 1km), AI 추천 개원 위치 3곳(50m 원)을 확인합니다. 시작점 이동 기능으로 위치를 미세 조정할 수도 있습니다.

3

지도 탐색

지도 탐색은 전국의 모든 분석 가능 지역을 지도 위에 점수와 함께 표시하는 핵심 기능입니다. HIRA 실제 치과 데이터(19,500+)를 기반으로 각 지역의 치과를 지도에 표시합니다.

● 지도 마커 색상

  • 파란색 — 80점 이상 (높음)
  • 초록색 — 70~79점 (양호)
  • 노란색 — 60~69점 (보통)
  • 빨간색 — 60점 미만 (낮음)

● 좌측 패널 기능

  • 검색 — 지역명, 구/군명, 시/도명, 장소명으로 검색 (카카오 지오코딩)
  • 필터 — 종합/인구/경쟁/비용/접근성/성장성 기준 정렬
  • 지역 목록 — 점수순으로 정렬된 지역 카드 클릭 시 상세 보기
  • 뒤로 가기 — 상세 보기에서 전체 지역 목록으로 복귀

지도 탐색 사용법

1

상단 메뉴에서 「지도 탐색」을 클릭합니다.

2

전국 지도에 표시된 점수 마커를 확인합니다. 숫자가 클수록 개원에 유리한 입지입니다.

3

좌측 패널의 필터 버튼을 이용하여 관심 있는 기준(인구, 경쟁, 비용 등)으로 정렬합니다.

4

지역 마커 또는 좌측 카드를 클릭하면 해당 지역의 주변 치과와 추천 개원 위치가 지도에 표시됩니다.

5

좌측 패널에서 「상세 분석 보기」 버튼을 클릭하면 입지분석 페이지로 이동합니다.

카카오 지오코딩 검색

검색창에 구체적인 장소명을 입력할 수 있습니다. 예: 「강남역」, 「인천경찰청」, 「판교 현대백화점」, 「수원 영통구」 등. 카카오 Local API가 정확한 GPS 좌표로 변환하여 해당 위치를 중심으로 분석합니다. 기존 데이터에 없는 지역도 AI가 자동으로 분석 데이터를 생성합니다.

4

입지분석

입지분석은 특정 지역의 개원 적합도를 5가지 축으로 심층 분석하는 기능입니다. 카카오 지오코딩으로 정확한 좌표를 찾고, HIRA 실제 치과 데이터로 반경 1km 내 치과를 분석합니다.

입지분석 페이지 구성

종합 점수 & 등급 (+ 지역 컨텍스트)

5축 분석의 가중 평균으로 계산된 종합 점수와 등급(A/B/C/D)을 표시합니다. 지역명 뒤에 (시도 시군구) 형태의 광역 지역 정보가 자동 표시됩니다.

5축 레이더 차트

인구/수요, 경쟁 강도, 임대 비용, 접근성, 성장성의 5가지 축을 시각적으로 비교합니다.

지역 상세 정보

인구수, 연령대 비율, 유동인구, 치과 수, 1인당 치과 비율, 평균 임대료 등의 데이터를 표시합니다.

입지 분석 지도 (70vh, 시작점 이동)

반경 1km 내 실제 치과를 지도에 표시합니다(HIRA 데이터). 지도 높이는 화면의 70%이며, 우측 상단에 범례가 기본 표시됩니다. 시작점 이동 버튼을 클릭하면 분석 중심 마커를 드래그하여 원하는 위치로 이동하고, 「이 위치에서 재분석」 버튼으로 즉시 재분석할 수 있습니다.

AI 추천 개원 위치 3곳

밀집도×틈새(density×gap) 알고리즘으로 산출한 최적 위치 3곳을 반경 50m 원으로 표시합니다. 각 위치별 추천 개원 규모(면적, 체어 수, 직원 수)와 예상 수익(일평균 환자, 월매출, 연매출, 손익분기점)을 제공합니다.

투자 시뮬레이션

초기 투자금, 월 고정비, 예상 매출을 시뮬레이션합니다.

분석 리포트 만들기

버튼 클릭 시 새 창에서 AI 입지분석 리포트가 열리며, 생성된 리포트는 자동으로 리포트 목록에 저장됩니다.

입지 분석 지도 화면 예시 지도 높이: 화면의 70%

입지분석 접근 방법

  • A. 홈페이지 검색창에서 지역명 또는 장소명을 입력하고 「AI 분석 시작」을 클릭
  • B. 지도 탐색에서 지역 클릭 후 좌측 패널의 「상세 분석 보기」를 클릭
  • C. 상단 네비게이션의 「입지분석」 메뉴를 클릭 후 지역 직접 입력
  • D. 입지분석 지도에서 「시작점 이동」 버튼을 클릭하고 마커를 드래그한 후 「이 위치에서 재분석」 클릭

AI 입지분석 리포트

AI 입지분석 리포트는 분석 결과를 종합하여 개원 의사결정에 필요한 모든 정보를 하나의 보고서로 제공합니다. 입지분석 페이지에서 「분석 리포트 만들기」를 클릭하면 새 창에서 리포트가 열립니다. 리포트 제목에는 지역명과 함께 (시도 시군구) 광역 지역 정보가 자동 표시됩니다.

06

인구 통계 & 경쟁 현황

핵심 리스크 요인과 기회 요인을 심각도와 함께 분석

리포트 생성 방법

  • 1. 입지분석 페이지에서 「분석 리포트 만들기」 클릭
  • 2. 새 창(새 탭)에서 리포트가 자동 생성
  • 3. 리포트가 자동으로 리포트 목록에 저장
  • 4. 브라우저 인쇄(Ctrl+P)로 PDF 다운로드 가능

리포트 활용법

  • - PDF 다운로드하여 인쇄 가능 (페이지 번호 자동 포함)
  • - 은행 대출 상담 시 사업계획 자료로 활용
  • - 부동산 중개인과 입지 상담 시 참고 자료
  • - 동업자 또는 투자자에게 분석 결과 공유
  • - 여러 지역 리포트를 비교하여 최종 결정

유의사항

AI 입지분석 리포트의 분석 결과는 공공 데이터 기반의 참고 자료입니다. 최종 개원 결정은 현장 답사, 전문가 상담 등을 종합적으로 고려하여 판단하시기 바랍니다.

추천 개원 위치

HIRA 실제 치과 데이터를 기반으로 밀집도×틈새(density×gap) 알고리즘이 해당 지역 내에서 개원에 가장 유리한 위치 3곳을 추천합니다.

밀집도×틈새(density×gap) 알고리즘

단순히 치과가 없는 빈 곳이 아니라, 수요가 검증된 상권(밀집도 높음) 내에서 가장 가까운 기존 치과와의 거리가 먼 지점(틈새가 큰 곳)을 찾습니다.

score = density(반경 300m 내 치과 수) × min(최근접 치과 거리, 300m)

밀집도가 높을수록, 그리고 가장 가까운 치과와의 거리가 멀수록 점수가 높아집니다.

알고리즘 원리 — 단순 공백 vs 밀집도×틈새
X 단순 빈 곳
O 밀집도 x 틈새
1

1순위 (주황색)

밀집도×틈새 점수가 가장 높은 최적 위치. 개원 최우선 검토 대상입니다.

2

2순위 (보라색)

1순위와 100m 이상 떨어진 차선 위치. 1순위 임대 물건이 없을 때 대안입니다.

3

3순위 (초록색)

2순위와도 100m 이상 떨어진 대안 위치. 세 곳 모두 검증된 상권 내 위치입니다.

추천 위치 3곳 — 지도 표시 예시 (반경 50m 원)

추천 위치별 사업 지표

각 추천 위치별로 주변 치과 평균과 틈새 보너스를 반영한 사업 지표가 제공됩니다.

추천 개원 규모

  • - 추천 면적 (평수)
  • - 추천 체어 수
  • - 추천 직원 수
  • - 예상 월임대료

예상 수익

  • - 일평균 환자 수
  • - 월매출 / 연매출
  • - 월순이익
  • - 초기투자금 / 손익분기 개월수

반경 50m 원 표시

추천 위치는 지도에서 반경 50m 원으로 표시됩니다. HIRA 데이터의 GPS 좌표에 20~50m의 오차가 있을 수 있어, 정확한 한 지점이 아닌 '이 영역 근처'라는 의미로 원형 구역을 표시합니다. 실제 개원 시 해당 반경 내에서 임대 물건을 탐색하시면 됩니다.

투자 시뮬레이션

치과 개원에 필요한 초기 투자금과 월 운영비, 예상 매출을 시뮬레이션하여 투자 대비 수익성을 분석합니다.

시뮬레이션 항목

초기 투자 비용

  • - 인테리어 공사비
  • - 의료장비 (유닛, X-ray 등)
  • - 보증금 / 권리금
  • - 인허가 비용
  • - 초기 재료비

월 고정 비용

  • - 임대료
  • - 인건비 (의사, 위생사, 직원)
  • - 재료비
  • - 관리비 / 공과금
  • - 마케팅비

예상 수익

  • - 월 예상 환자 수
  • - 보험 급여 매출
  • - 비급여 매출
  • - 순이익 예상치
  • - 손익분기점 (BEP)

슬라이더 조절

입지분석 페이지 하단의 투자 시뮬레이터에서 슬라이더를 움직여 면적, 체어 수, 인테리어 등급 등을 조절하면 실시간으로 예상 비용과 수익이 변경됩니다.

PDF 인쇄 가이드

AI 입지분석 리포트는 브라우저의 인쇄 기능을 이용하여 PDF로 저장할 수 있습니다.

PDF 저장 방법

1

리포트 상세 페이지에서 Ctrl + P (Mac: Cmd + P)를 누릅니다.

2

인쇄 대상을 「PDF로 저장」으로 선택합니다.

3

용지 크기 A4, 방향 세로를 확인합니다.

4

「저장」 버튼을 클릭하면 PDF 파일이 다운로드됩니다.

PDF 인쇄 최적화 기능

  • ✓ 헤더/푸터 자동 숨김 (깔끔한 출력)
  • ✓ 페이지 번호 자동 표시 (오른쪽 하단, 예: 1 / 5)
  • ✓ A4 용지에 최적화된 여백 설정
  • ✓ 섹션별 페이지 나눔 (내용이 잘리지 않음)
  • ✓ 그래프/차트 색상 정확히 인쇄
  • ✓ 지도 영역 400px 고정 높이로 출력

인쇄 설정 권장값

  • 용지 크기: A4
  • 방향: 세로 (Portrait)
  • 여백: 기본값 사용
  • 배경 그래픽: 켜기 (색상 차트 출력을 위해 필수)
  • 머리글/바닥글: 끄기 (리포트에 자체 페이지 번호 포함)
*

인쇄 팁

Chrome 브라우저에서 인쇄 시 「배경 그래픽」 옵션을 반드시 활성화하세요. 이 옵션이 꺼져 있으면 차트 색상과 배경이 출력되지 않습니다. 페이지 번호는 리포트 자체에 포함되어 있으므로 브라우저의 머리글/바닥글은 끄는 것을 권장합니다.

12

활용 팁

Tip 1 구체적 장소명으로 검색하기

「서울 강남구」 같은 넓은 지역 대신 「강남역」, 「인천경찰청」, 「판교 현대백화점」 등 구체적인 장소명을 입력하세요. 카카오 지오코딩이 정확한 좌표를 찾아 해당 위치를 중심으로 분석합니다.

Tip 2 시작점 이동으로 미세 조정

검색 결과 위치가 원하는 곳과 약간 다르다면 「시작점 이동」 버튼을 클릭하고 마커를 드래그하세요. 건물 단위, 블록 단위의 미세한 위치 조정이 가능하며, 이동 후 「이 위치에서 재분석」으로 즉시 새로운 분석 결과를 확인할 수 있습니다.

Tip 3 추천 위치 3곳 비교하기

AI가 추천하는 3곳의 사업 지표(월매출, 손익분기점 등)를 비교하세요. 1순위가 항상 최선은 아닙니다 — 임대료가 저렴한 2순위나 3순위가 손익분기점이 더 빠를 수 있습니다.

Tip 4 여러 지역 리포트 비교

관심 지역 3~5곳의 입지분석 리포트를 각각 생성하고, 리포트 목록에서 여러 리포트를 동시에 열어 비교하세요. 각 리포트가 새 창에서 열리므로 나란히 비교가 가능합니다.

Tip 5 50m 원 내에서 임대 물건 탐색

추천 위치의 반경 50m 원이 실제 개원 후보지 탐색 범위입니다. 해당 반경 내의 상가 건물을 현장 답사하고, 부동산 중개인에게 이 영역의 임대 물건을 문의하세요.

Tip 6 경쟁 치과 상세 분석

HIRA 데이터로 표시되는 주변 치과의 의사 수, 전문의 수, 개설 연도를 파악하여 차별화 전략을 세우세요. 예를 들어 주변에 오래된 치과만 있다면 최신 시설로 차별화할 수 있습니다.

Tip 7 성장성 지표 주목

현재 점수가 높은 지역보다 성장성 점수가 높은 지역이 장기적으로 유리할 수 있습니다. 신규 아파트 단지, 개발 계획이 있는 지역을 주목하세요.

Tip 8 리포트를 대출 상담에 활용

AI 입지분석 리포트를 PDF로 출력하여 은행 대출 상담 시 사업 계획 보충 자료로 활용하세요. 페이지 번호가 자동 포함되어 전문적인 보고서로 제출할 수 있습니다.

Tip 9 리포트 목록으로 이력 관리

생성한 리포트는 자동으로 목록에 저장됩니다. 시간이 지난 후에도 「리포트」 메뉴에서 이전에 분석한 지역을 다시 열어볼 수 있으므로, 의사결정 과정을 체계적으로 관리하세요.

13

용어 사전

HIRA 건강보험심사평가원. 전국 치과의원·치과병원의 공식 데이터(GPS, 의사 수, 전문의 수, 개설일 등)를 제공하는 공공기관.
카카오 지오코딩 카카오 Local API를 이용하여 장소명(예: '강남역')을 정확한 GPS 좌표(위도·경도)로 변환하는 기술.
밀집도×틈새 density×gap 알고리즘. 치과 밀집도(수요 검증)와 최근접 치과 거리(경쟁 공백)를 곱하여 최적 위치를 산출하는 추천 알고리즘.
반경 50m 원 추천 위치를 GPS 오차(20~50m)를 감안하여 점이 아닌 원형 영역으로 표시하는 방식.
종합 점수 5축 분석의 가중 평균 점수. 100점 만점으로 개원 적합도를 나타냄.
5축 분석 인구/수요(25%), 경쟁 강도(25%), 임대 비용(20%), 접근성(15%), 성장성(15%)의 5가지 평가 기준.
인구당 치과 비율 지역 인구수를 치과 수로 나눈 값. 숫자가 클수록 경쟁이 적음.
유동인구 하루 동안 해당 지역을 통과하거나 방문하는 인구 수.
상권 등급 A(최상) ~ C(보통) 등급으로 나뉘는 상업 활성화 수준 지표.
경쟁 위협도 특정 치과가 신규 개원 치과에 미치는 경쟁 압박 정도 (높음/보통/낮음).
비급여 비율 전체 진료 중 건강보험 비적용(자비 부담) 진료의 비율.
BEP (손익분기점) 매출이 총비용과 같아지는 시점. 이후부터 순이익 발생.
